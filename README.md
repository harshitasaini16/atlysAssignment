# Atlys Assignment

This repository contains the solution for the Atlys assignment. It demonstrates the implementation of Function chain calculator.


## Installation

Follow the steps below to set up and run the project locally.

1. Clone the repository:
   ```bash
   git clone https://github.com/harshitasaini16/atlysAssignment.git

2. cd atlysAssignment

3. npm install or yarn install

4. npm start

Your application will start on "http://localhost:3000/"
